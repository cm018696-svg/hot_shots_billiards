# Hot Shots Billiards Club

Website for a billiards club with table booking and admin management.

## Pages

- `index.html` - Main website
- `admin.html` - Staff dashboard for table management
- `reserva.html` - Reservation page
- `success.html` - Booking confirmation

## Deployment (GitHub Pages)

1. Create a new repository on GitHub
2. Push your code:
   ```bash
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/hot-shots-billiards.git
   git push -u origin main
   ```
3. Go to Settings > Pages
4. Select "main" branch and click Save

Your site will be available at: `https://YOUR_USERNAME.github.io/hot-shots-billiards/`

## Local Development

Open `index.html` in your browser, or use a local server:
```bash
npx serve
```
